# 🚺 Women Safety Application

> A smart and reliable mobile application designed to enhance women's safety using real-time technology and instant emergency features.

---

## 📱 Overview

The **Women Safety Application** is built to provide quick assistance during emergencies.  
It enables users to send **instant alerts**, share **live location**, and access **help features** with just a single tap.

---

## ✨ Features

- 🚨 **SOS Emergency Alert**
  - Sends instant alerts to saved contacts

- 📍 **Live Location Tracking**
  - Shares real-time location with trusted contacts

- 📞 **Quick Call Feature**
  - Direct call to emergency contacts

- ⚡ **Fast & User-Friendly UI**
  - Simple and intuitive interface for quick access

- 🔒 **Secure & Reliable**
  - Ensures user safety and data privacy

---

## 🛠️ Tech Stack

| Technology | Usage |
|----------|------|
| Flutter | Frontend (Mobile UI) |
| Firebase | Backend & Real-time Database |
| Google Maps API | Location Tracking |
| Android Studio | Development Environment |

## 🚀 Installation & Setup

### 🔧 Prerequisites
- Flutter SDK installed  
- Android Studio / VS Code  
- Firebase Project Setup  

---

### ⚙️ Steps

```bash
# Clone the repository
git clone https://github.com/buddherohit/Women-Safety.git

# Navigate to project folder
cd Women-Safety

# Install dependencies
flutter pub get

# Run the app
flutter run
